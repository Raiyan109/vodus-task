# Hiking Gear Inventory Website with JQuery

This simple web application displays a list of hiking gear items in card format, allowing users to click a "Details" button to view more information about each item. The application uses jQuery to fetch and display hiking gear data from an external source.

## How to Run the Application

1. Clone or download this repository to your local machine.

2. Open the `index.html` file in a web browser (e.g., Google Chrome, Firefox, or Safari).

3. The hiking gear items will be displayed in card format with a "Details" button for each item.

4. Click the "Details" button to view additional information about each item. The "Details" section will expand and collapse when the button is clicked.

5. Enjoy browsing the hiking gear inventory!

## Dependencies

- jQuery (included via a Content Delivery Network)

## Data Source

The application fetches hiking gear data from an external source (URL: https://hiking-warehouse-server.vercel.app/api/items).
